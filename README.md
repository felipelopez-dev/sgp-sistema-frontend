# sgp-sistema-frontend
Desenvolvido para resolver um problema real na minha empresa, o Software de Gerenciamento de Produção (SGP) é uma ferramenta web que calcula o saldo de caixa diário de forma rápida e precisa. Criado com Html, Css e JavaScript, o SGP oferece uma interface intuitiva para inserir dados de produção e obter resultados precisos em poucos cliques.
